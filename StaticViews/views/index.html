Welcome to Modulr!
{{"include": "StaticViews/img/modulr-stroke.svg"}}

Modulr is code stipulation utility written in .NET 5, primarily for Java.

Using Docker and custom unit testers, Modulr is capable of safely testing code with minimal risk of host instability.

Compared to other utilities like Stipulator and Antistipulator, code can be safely tested on the internet without installation on the client-side.

This maks it suitable for students using Chromebooks or other devices with limited installation ability.

Modulr is configurable.

Configuration files and web-based managers allow admins to configure safe compilation.

Files are associated with each test, making migration easy.

Modulr is compatible with MySQL and SQL Server, making it installable on many servers.

Modulr is open source.

No secrets: password storage algorithms, Docker images, and unit-testing tools are on my GitHub to view.

Python support and SQLite storage are also planned in the future.
